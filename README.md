# 웹 지뢰찾기

브라우저에서 바로 실행할 수 있는 지뢰찾기 게임입니다. 난이도를 선택해 안전한 칸을 모두 열고 지뢰를 피하세요.

## 주요 기능

- 초급/중급/고급 난이도(9×9/16×16/16×30) 지원
- 첫 번째 클릭은 항상 안전한 영역에서 시작
- 남은 지뢰 수, 경과 시간, 열린 칸 수 실시간 표시
- 데스크톱은 오른쪽 클릭, 모바일은 "깃발 모드" 전환으로 깃발 배치
- 같은 숫자 칸을 두 번 클릭하면 주변 깃발 수에 맞춰 일괄 오픈

## 실행 방법

1. 이 저장소를 내려받거나 ZIP으로 추출합니다.
2. `index.html` 파일을 브라우저(Chrome, Safari 등)에서 열면 바로 게임을 즐길 수 있습니다.

> 별도의 빌드나 패키지 설치 과정이 필요하지 않습니다.

## 구조

- `index.html`: 게임 마크업과 기본 UI
- `styles.css`: 반응형 스타일과 색상 테마
- `script.js`: 보드 생성, 지뢰 배치, 승패 판정 등 게임 로직

즐거운 지뢰찾기 플레이 되세요!
